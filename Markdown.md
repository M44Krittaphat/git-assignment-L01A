# Markdown คืออะไร?
Markdown เป็น Lightweight Markup Language ที่ใช้สำหรับจัด Format ของ Plain Text บน Document และแปลงเป็น HTML บนเว็บต่างๆ มีจุดเด่นที่ความง่ายต่อการอ่านและการเขียน เพราะมีหน้าตาคล้ายกับภาษาที่เราใช้กัน โดย Markdown มีนามสกุลไฟล์เป็น .md
# Markdown เขียนยังไง?
## Headings
เราจะใช้ # สำหรับทำ Headings ทั้ง 6 แบบ โดยเพิ่มจำนวน # ตามเลเวล Heading โดยที่จำนวน # ที่น้อยจะเป็นสำหรับหัวข้อที่ใหญ่และ # ที่มากมีไว้สำหรับหัวข้อย่อยตามลำดับ
```
# Heading 1 
## Heading 2 
### Heading 3 
#### Heading 4 
##### Heading 5 
###### Heading 6
```
ผลลัพท์

# Heading 1 
## Heading 2 
### Heading 3 
#### Heading 4 
##### Heading 5 
###### Heading 6
## Paragraphs (New Line)
ตอนแสดงผล ข้อความที่บรรทัดติดกันจะกลายเป็นบรรทัดเดียวกันในตอนแสดงผล ซึ่งถ้าหากเราต้องการเว้นบรรทัดให้ข้อความ เราจะต้องเคาะเว้นลงมาเป็นบรรทัดเปล่า 1 บรรทัดซะก่อน เรียกว่า Hard Break เช่น
```
Hello World

Nice to meet you
```
ได้เป็นเป็น

Hello World

Nice to meet you

และอีกวิธีที่ใช้ขึ้นบรรทัดใหม่ได้ ก็คือเคาะเว้นวรรค 2 ครั้ง หลังบรรทัดที่ต้องการให้ขึ้นบรรทัดใหม่ ซึ่งแบบนี้เรียกว่า Soft Break เช่น
```
Hello World  
Nice to meet you
```
ได้เป็นเป็น

Hello World  
Nice to meet you

ซึ่งจะมีการเรียงของบรรทัดที่ติดกันมากกว่า Hard Break อีกด้วย
## Horizontal Lines
Horizontal Lines
นอกจาก — จะใช้แบ่งเลเวล Heading ได้แล้ว ยังใช้สร้างเส้นคั่นแนวนอนได้อีกด้วย  
ตัวอย่าง
```
---
```
ผลลัพธ์ที่ได้

---
ถ้าเหนือเส้นคั่นมีข้อความ เราต้องเว้นบรรทัดลงมาก่อนนะ เส้นคั่นถึงจะแสดง ไม่อย่างนั้น จะทำให้ข้อความข้างบนเป็น Heading แทน

ตัวอย่างกรณีที่ใช้เส้นคั่นโดยไม่เว้นบรรทัดข้อความ
```
ข้อความ
---
```
ผลลัพธ์ที่ได้

ข้อความ
---
เราจะแก้ปัญหาจากตัวอย่างที่ผ่านมาได้ด้วยการเว้นบรรทัด

เช่น
```
ข้อความ

---
```
ผลลัพธ์ที่ได้

ข้อความ

## Text Styles
Text Styles เป็นวิธีจัดการกับข้อความให้มีความสวยงามและอ่านง่ายมากขึ้น โดยที่เราจะทำตัวเอียงและตัวหนา โดยใช้ _ หรือ * ครอบข้อความที่เราต้องการให้เอียง และใช้ __ หรือ ** ครอบข้อความที่เราต้องการให้หนา

### Italics And Bold
ตัวอย่างที่
```
นี่คือ _ตัวเอียง_

นี่คือ __ตัวหนา__

นี่คือ *ตัวเอียง* เหมือนกัน

นี่คือ **ตัวหนา** เหมือนกัน
```
ผลลัพท์

นี่คือ _ตัวเอียง_

นี่คือ __ตัวหนา__

นี่คือ *ตัวเอียง* เช่นกัน

นี่คือ **ตัวหนา** เช่นกัน

แล้วเราอยากได้ทั้งตัวหนาและตัวเอียงพร้อมกันล่ะ? ก็ให้ใช้ทั้งคู่พร้อมๆกันโดยให้ตัวใดัวหนึ่งครอบอีกตัว

เช่น
```
นี่คือข้อความที่เป็นทั้ง ***ตัวหนาและตัวเอียง***

นี่คือข้อความที่เป็นทั้ง ___ตัวหนาและตัวเอียง___ เช่นกัน

นี่คือข้อความที่เป็นทั้ง **_ตัวหนาและตัวเอียง_** เช่นกัน
```

นี่คือข้อความที่เป็นทั้ง ***ตัวหนาและตัวเอียง***

## Lists
### Unordered Lists

```
วัตถุดิบทำพิซซ่า

+ แป้งพิซซ่า
- ชีส

```
